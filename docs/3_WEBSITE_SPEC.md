## Функционал

### Калькулятор доставки
- Поля: страна (dropdown), вес (кг), Д×Ш×В (см), чекбокс страховки
- Объёмный вес = Д×Ш×В / 5000 → расчётный = max(реальный, объёмный)
- Результат: вес, стоимость, срок, CTA → ЛК

### Трекинг
- Ввод трекинг-номера без регистрации → текущий статус
- Полная история — в ЛК

### Форма "Написать менеджеру"
- Поля: имя, страна, вопрос, предпочтительный мессенджер
- Авто-ответ после отправки

## SEO-требования

### Технический SEO
- Sitemap.xml (авто Next.js), robots.txt, canonical, Open Graph
- Schema.org: `LocalBusiness`, `FAQPage`, `BreadcrumbList`, `Service`
- PageSpeed ≥ 90 (mobile), LCP < 2.5s, FID < 100ms, CLS < 0.1

### Ключевые слова

**Высокочастотные (главная):**
доставка из сша в снг, карго из сша, отправить посылку из америки

**Средне-частотные (страницы стран):**
доставка из сша в [россию/беларусь/кыргызстан/казахстан/узбекистан], карго сша [страна], сколько стоит доставка из сша в [страна]

**Нишевые:**
доставка эхолота из сша, карго без санкционных ограничений, склад в делавэре для заказов, пункт приема посылок [город] снг

## Дизайн-требования

- **Стиль:** современный, минималистичный, доверительный
- **Цвета:** slate/серый + синий акцент, тёплый акцент для CTA
- **Шрифт:** Geist · **Иконки:** Heroicons/Lucide/Phosphor · **Флаги:** SVG
- **Отзывы:** скриншоты в рамке "мессенджер-стиле"
- **Карта:** Google Maps / Mapbox, маркеры с логотипом
- **Анимации:** счётчики Counter Up, fade-in карточки. Максимум 1–2 на секцию.
- **Мобайл-приоритет:** таблицы → горизонтальный скролл или карточки, sticky CTA-бар, hero без скролла
