## О компании

**OWAY Cargo** — карго с корнями в СНГ-диаспоре. Основана в 2022 (Стамбул) **Дастаном Турумбековым** — бывшим байером для предпринимателей СНГ. Клиенты спрашивали _"Как отправить это домой?"_ — так появился OWAY Cargo.

С 2023 — работаем из США. Фокус: USA → CIS (РФ, РБ, КГ, КЗ, УЗ). Также работаем с Китаем, Кореей, Германией, Италией, Турцией.

1. **Точки в диаспорных районах** — 15 городов сейчас, 37 к сентябрю 2025. Барбершопы, магазины, мастерские — там, где живут наши люди.

3. **Товары без ограничений** — электроника, эхолоты, тактика, коммерческая техника. Нельзя только: оружие, наркотики, взрывчатка.
4. **Цены от $12/кг, без минимума** — подробные тарифы → [2_BUSINESS.md](2_BUSINESS.md)
5. **Часть комьюнити** — спонсируем мероприятия диаспоры по всей Америке. Нас знают лично.

## Наши клиенты

**Тип А — Отправитель в США**
Из СНГ, живёт в Америке, отправляет домой (технику, одежду, подарки, коммерческий товар).
_Боль:_ далеко ехать / не берут товар / дорого → _Ответ:_ точка рядом, без ограничений, хорошая цена.

**Тип Б — Заказчик из СНГ**
Хочет купить из США (частное лицо или бизнес для маркетплейса).
_Боль:_ нет карты USA / не знает как / боится таможни → _Ответ:_ адрес в Делавэре, выкуп, менеджер, оплата в местной валюте.
