## Идеи новых продуктов

### 1. Калькулятор таможни (приоритет: максимальный)
Вкладка в Mini App: тип товара + стоимость + страна → порог, пошлина, итог.
Ни один конкурент не предлагает. Высокий viral — "сохрани бот, сэкономит деньги".

### 2. Price Alert бот (@OwayPriceBot)
Клиент присылает ссылку Amazon/Nike → бот следит за ценой → при падении: "Подешевел! С OWAY в Бишкек: ~$180 total."
Бесплатно 3 товара, платно 100 сом/мес до 20.

### 3. Витрина "Что выгодно везти"
Вкладка в Mini App: карточки товаров с расчётом (цена USA + доставка vs цена в СНГ = выгода).

### 4. Шеринговые tracking-ссылки
`owaycargo.com/track/OWY-2025XXXXX` — получатель шарит маме, та видит статус без установки. Каждая ссылка = бренд-импрессия.

### 5. Сообщество покупателей
Telegram-группа: делятся покупками, задают вопросы, распаковки. OWAY как эксперт.

### 6. Структурированный шопинг-помощник
Вместо текстового запроса → flow: категория → бренд → размер → бюджет → оценка стоимости → подтверждение.
